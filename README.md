# 📘 Understanding JavaScript and Beyond

This repository is my personal journey to deeply understand JavaScript—not just how it works, but *why* it works.

The goal is to move beyond memorization and build true intuition by breaking concepts down, rebuilding them, and applying them in real scenarios.

---

## 🧠 Learning Philosophy

This repo follows a structured way of learning:

### 1. 🔲 Black Box (Big Picture)

* Understand the purpose of a concept
* Treat it as a "black box" first
* Focus on *what it does*, not how

---

### 2. 🧩 Breakdown (Understand the Parts)

* Take the concept apart
* Study each piece:

  * in context
  * and on its own

---

### 3. 🔁 The Loop (Deep Understanding)

For each concept:

* **Care** → Why does this matter?
* **Invent** → Try to recreate it from scratch
* **Absorb** → Study the real implementation and compare

---

### 4. 🔥 Reinforce (Make it Stick)

* **Practice** → Apply it to real problems
* **Explain** → Write it in your own words
* **Explore** → Modify it, break it, experiment

```

   ↓
var
   ↓
Hoisting (initialized as undefined)
   ↓

   ↓
```
